Order Management Guide

Create, track, and manage customer orders efficiently

Quick Navigation
Create Order Track Order Order Status Payments
Important

Efficient order management is crucial for customer satisfaction. Keep orders organized and update statuses regularly.

Creating a New Order

1

Go to Orders

Click on the Orders menu in the main navigation.

2

Click "Create New Order"

Click the green "New Order" button in the top right corner.

3

Select or Create Customer

  • Select an existing customer, or
  • Click "Add New Customer" to create one
  • Fill in customer details: Name, Email, Phone, Address
4

Add Products to Order

  • Click "Add Item"
  • Search and select products from your catalog
  • Enter quantity needed
  • Price is automatically calculated
  • Repeat for additional products
5

Set Order Details

  • Order Date: When order is placed (defaults to today)
  • Delivery Date: When customer wants it
  • Notes: Special instructions or requirements
  • Discount: Apply if applicable
6

Select Payment Method

  • Cash
  • Credit Card
  • Check
  • Bank Transfer
  • Other
7

Review and Save

Review the order summary and click "Create Order" to save.

Pro Tip

Use the "Save as Draft" option if you need to come back and complete the order later.

Order Status Guide

Understanding order statuses helps you manage your workflow effectively.

Order Status Flow
Status Description Action Required
Draft Order saved but not confirmed Complete and confirm order
Pending Order confirmed, awaiting processing Prepare order items
Processing Order is being prepared Continue preparation
Ready Order is ready for pickup/delivery Arrange pickup or delivery
Shipped Order is out for delivery Track and confirm delivery
Delivered Order successfully delivered Confirm receipt with customer
Cancelled Order cancelled by customer or system Process refund if needed

Updating Order Status:

1

Open the order you want to update

2

Click on the status dropdown at the top of the order

3

Select the new status from the list

4

Add notes if needed and click "Update Status"

Tracking Orders

Finding a Specific Order:

1
Go to Orders List

Navigate to Orders in the main menu.

2
Use Search or Filter
  • Search by Order ID
  • Search by Customer Name
  • Filter by status, date range, or payment status
3
Click to View Details

Click on an order to see full details, items, timeline, and notes.

Order Timeline:

Each order shows a complete timeline of all actions and status changes. You can:

  • See when status changed
  • View who made changes
  • Read associated notes
  • Track all communications

Payment Management

Recording Payments:

1

Open the order

2

Click "Record Payment" in the payment section

3

Enter payment details:

  • Amount paid
  • Payment method
  • Transaction reference/check number
  • Payment date
4

Click "Save Payment"

Payment Status:

  • Not Paid: Order hasn't been paid yet
  • Partially Paid: Customer paid part of the order
  • Paid: Order has been fully paid
  • Refunded: Payment was refunded to customer
Important

Always record payments immediately to keep financial records accurate and up-to-date.

Advanced Features

Recurring Orders

Set up orders that repeat automatically (e.g., weekly bread delivery)

Setup: Create order → Select "Recurring" → Choose frequency

Print Orders

Print order invoices or labels for kitchen/packing

Use: Open order → Click Print → Choose format

Send Notifications

Send order updates to customers via email or SMS

Use: Open order → Click "Send Notification"

Duplicate Order

Create new order using items from previous order

Use: Open order → Click "Duplicate"

Best Practices

DO's
  • ✓ Update order status regularly
  • ✓ Record payments immediately
  • ✓ Add delivery notes for special requests
  • ✓ Confirm delivery with customers
  • ✓ Keep order history for records
DON'Ts
  • ✗ Forget to update status
  • ✗ Skip payment recording
  • ✗ Create duplicate orders
  • ✗ Delete completed orders
  • ✗ Miss delivery dates
